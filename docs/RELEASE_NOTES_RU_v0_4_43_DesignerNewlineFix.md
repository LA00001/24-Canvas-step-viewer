# Inventor IPT Organizer v0.4.43 — DesignerNewlineFix

## Что исправлено

Исправлена ошибка компиляции из `Form1.Designer.cs`:

- `CS1010: Символ новой строки в константе`
- `CS1002: Требуется ";"`.
- `CS1040: Перед директивами препроцессору могут находиться только пробельные знаки`.

Причина: текст второй кнопки был записан как реальный перенос строки внутри C# string literal.

## Исправление

Теперь строка записана так:

`this.ButtonIptCreateFeatureFolder2.Text = "Create feature\r\nbrowser folder #2";`
