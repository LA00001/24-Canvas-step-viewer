# Path policy

Starting with v0.4.32 package names and paths are shortened to avoid Windows MAX_PATH issues.

Archive name example:
- `InventorIptOrg_v0_4_32.zip`

Expected extracted layout:
- `InventorIptOrg.sln`
- `src/InventorIptOrg/InventorIptOrg.csproj`

No `GitHub package` in archive names.
No duplicated root folder inside the archive.
