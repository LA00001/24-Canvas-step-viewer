# v0.4.88 — FastPleasantViewer

## Почему эта версия

v0.4.85-v0.4.87 проверяли chunks / spatial chunks / LOD.  
Итог по логам: архитектура работает, но ощущение скорости стало хуже.

v0.4.88 возвращается к быстрой практичной viewer-базе v0.4.84.

## Что сделано

- Default render mode теперь `Surface`, а не `Surface+Edges`.
- `Surface` рисует лёгкую поверхность без внутренней ряби рёбер.
- `Surface+Edges` оставлен отдельным режимом.
- `FastDragWireframe` оставлен простым, без chunk/LOD overhead.
- FastDrag дополнительно прорежен: меньше линий во время вращения.
- В лог добавлено `ViewerProfile=FastPleasant`.

## Что НЕ включено

- v0.4.87 LOD boxes не используются.
- chunk selection не запускается на каждом кадре.
- BoxGridCore остаётся доступным как слой/overlay, но не тормозит STEP drag.

## Цель

Быстрый и приятный viewer:

- меньше ряби;
- меньше логики на mouse move;
- быстрое вращение;
- качественная полная поверхность после отпускания.
