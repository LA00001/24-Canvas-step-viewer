# Inventor IPT Organizer v0.4.41 — FeatureFolderButton2

## Что сделано

Добавлена вторая кнопка рядом с `Create feature browser folder`:

`Create feature browser folder #2`

Она является простой копией первой кнопки и вызывает тот же обработчик:

`ButtonIptCreateFeatureFolder_Click`

То есть создаёт BrowserFolder из текущего списка `Features / элементы` тем же проверенным путём.
