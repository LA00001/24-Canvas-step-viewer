# Inventor IPT Organizer v0.4.34 — TraceDepthFix

## Что исправлено

После нажатия `Custom rect select` желто-зелёный trace box выглядел слишком тонким по глубине.

Причина: ветка RealCamera строила trace box как тонкий слой вокруг camera target plane.

## Что сделано

- trace box по-прежнему считает экранные координаты через RealCamera + aspect correction;
- скрытая ось теперь берётся не как тонкая толщина, а как полный диапазон `Bounds` всех `TouchedCells`;
- визуальный marker теперь должен занимать полную глубину всех задействованных spatial-cubes.
