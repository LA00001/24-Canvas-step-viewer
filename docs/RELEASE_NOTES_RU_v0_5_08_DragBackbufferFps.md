# v0.5.08 — DragBackbufferFps

Build name: `DRAG_BACKBUFFER_FPS`

## Главное

Исправлено мерцание и исчезновение нормального отображения при вращении ЛКМ.

- Добавлен двойной буфер и явный 32-bpp backbuffer.
- Новый кадр показывается только после полного завершения рендера.
- Drag ограничен целевыми 30 FPS.
- Частые MouseMove объединяются, а не запускают отдельный синхронный рендер каждый раз.
- Вместо плотного `FastDragWireframe` используется уменьшенный shaded True Front кадр текущей ориентации.
- После отпускания ЛКМ автоматически строится полный качественный кадр.
- В правом верхнем углу показываются FPS, frame time и dropped/coalesced input count.
- Добавлена диагностика drag-сессии в лог.

`Mode: Mesh True Front — v0.5.06`
