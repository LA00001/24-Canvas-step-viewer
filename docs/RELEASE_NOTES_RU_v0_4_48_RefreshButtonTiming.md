# Inventor IPT Organizer v0.4.48 — RefreshButtonTiming

## Что добавлено

Две кнопки Refresh в красной рамке теперь после каждого запуска показывают время последней работы прямо в названии кнопки.

## Кнопки

- `Refresh browser tree`
- `Refresh legacy browser tree`

## Поведение

Во время выполнения: `working...`

После завершения:

`last: X.XXX s`

## Логи

- `BROWSER_TREE_REFRESH_BUTTON_SECONDS`
- `LEGACY_BROWSER_TREE_REFRESH_BUTTON_SECONDS`
