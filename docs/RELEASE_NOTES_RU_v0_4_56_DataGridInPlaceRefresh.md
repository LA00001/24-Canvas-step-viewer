# Inventor IPT Organizer v0.4.56 — DataGridInPlaceRefresh

## Что исправлено

`Refresh tree #2 BASE` больше не обязан пересоздавать все строки DataGridView.

## Как работает

1. Берётся names-only/cache snapshot.
2. Если текущая таблица уже имеет такое же количество строк, применяется in-place update.
3. Обновляются Level / Type / Name / row.Tag / ReadOnly.
4. X/Y/Z не очищаются.

## Зачем

v0.4.55 показал, что `PopulateIptBrowserTreeGridFromItems` занял около 8.47 с. v0.4.56 должен убрать эту цену на совпадающем 269-row refresh.
