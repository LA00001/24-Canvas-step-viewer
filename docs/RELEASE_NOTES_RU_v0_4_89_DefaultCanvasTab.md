# v0.4.89 — DefaultCanvasTab

## Что сделано

`.ipt canvas` теперь выбирается вкладкой по умолчанию при старте приложения.

## Почему

После v0.4.88 основная рабочая зона для STEP viewer — `.ipt canvas`, поэтому больше не нужно каждый раз вручную переключаться на эту вкладку.

## Что сохранено из v0.4.88

- default `Mode = Surface`;
- быстрый `FastDragWireframe`;
- без v0.4.87 chunk/LOD overhead;
- `ViewerProfile=FastPleasant`;
- локальный STEP viewer без внешнего открытия STEP.
