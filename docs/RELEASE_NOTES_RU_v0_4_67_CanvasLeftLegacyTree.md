# Inventor IPT Organizer v0.4.67 — CanvasLeftLegacyTree

## Что изменено

В левом нижнем углу `.ipt canvas` теперь не служебное дерево слоёв, а настоящее:

`Legacy browser tree / старое дерево браузера Inventor`

## Как обновлять

Кнопка:

`LR Refresh`

запускает `ButtonIptLegacyRefreshBrowserTree_Click` и обновляет постоянное дерево слева.

## Что осталось

- квадратные кнопки в левом верхнем углу;
- крупный preview справа;
- временные окна крупных областей.
