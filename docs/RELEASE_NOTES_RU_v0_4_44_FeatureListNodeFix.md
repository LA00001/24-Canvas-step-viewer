Исправлена ошибка компиляции:

## Что добавлено

Добавлен helper `GetBrowserNodesForSubfolderFromFeatureList`.

Он работает так:

1. Берёт выбранные строки из `Features / элементы`.
2. Если ничего не выделено, берёт весь текущий `Features-list`.
3. Для каждого `FeatureListItem` получает `BrowserNode` через `ModelBrowserPane`.
4. Возвращает дедуплицированный список для создания подпапки.
