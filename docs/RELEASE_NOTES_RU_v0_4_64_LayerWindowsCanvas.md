# Inventor IPT Organizer v0.4.64 — LayerWindowsCanvas

## Что добавлено

Добавлена новая вкладка `.ipt canvas`.

Цель: крупные области показывать по запросу в отдельных временных окнах, а не держать все таблицы постоянно на основной форме.

## Полотно

Слева:

- `Model preview / превью модели`
- статус Browser grid / Legacy / Spatial BASE / Fast cache

Справа:

- квадратные кнопки со значками
- ToolTip-подсказки с задержкой

## Временные окна

- Browser Truth Cache
- Spatial Geometry Cache
- Merged View Cache
- Fast Cache View
- Legacy Browser Tree
- Model Preview / Status

## Важно

v0.4.64 не заменяет канонический `Refresh browser tree` из v0.4.48.  
Он только добавляет отдельный слой диагностики/вывода крупных областей по запросу.

## v0.4.66 CANVAS_LAYOUT_LEFT_TOOLS

Перестроена вкладка `.ipt canvas` по замечаниям со скрина:

- квадратные кнопки перенесены в левый верхний угол;
- слева внизу добавлено постоянно открытое дерево слоёв;
- preview модели перенесён в крупную правую область и занимает весь оставшийся экран полотна;
- двойной клик по узлам дерева слоёв открывает соответствующие временные окна;
- ToolTip-подсказки с задержкой сохранены.
