## Новая кнопка

Добавлена кнопка:

`Show / Hide body RangeBoxes`

Она берёт тела из текущего visible-list и рисует для каждого:

`SurfaceBody.RangeBox`

как временную `ClientGraphics`-графику Inventor.

- зелёный: trace box рамки;
- случайные полупрозрачные цвета: RangeBox каждого выбранного тела.
